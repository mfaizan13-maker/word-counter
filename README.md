# 🏎️ Word Counter Pro — Bugatti Edition

Ek simple, tez aur khoobsurat **word counter** tool jo sirf browser me chalta hai (koi server nahi).

## ✨ Features
- 📊 Words, characters, sentences, paragraphs, numbers aur reading time ginta hai
- 🔑 **Keyword Density** — SEO ke liye sab se zyada aane wale words (% ke saath)
- 🔁 **Repetition Checker** — dohraaye gaye phrases dhoondta hai
- 🔠 **Case converter** — UPPERCASE / lowercase / Title Case
- 📋 **Copy results** — ek click me saare stats copy
- 🏎️ **Bugatti-inspired theme** — dark carbon + Bugatti blue

## 🚀 Kaise chalayein
`index.html` ko browser me kholein — bas!

- CSS
